## Usage


### Installation


```
npm i v3-boilerplate
``` 
OR 

```
yarn add v3-boilerplate
```




## Project setup


### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
